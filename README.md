## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/JMax45/r6sRandomizer`

### Setup

> move to the directory with the program

```shell
$ cd r6sRandomizer/build-r6sRandomizer-Desktop-Debug/
```

> run the program

```shell
$ chmod +x r6sRandomizer
$ ./r6sRandomizer
```
